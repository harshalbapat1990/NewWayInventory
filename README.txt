Inventory Management App

1. Extract the contents of the zip file to a folder on your computer.
2. Open the extracted folder.
3. Double-click the "InventoryAppBackend.exe" file to start the app.
4. Open your browser and navigate to http://127.0.0.1:5000 to access the app.

Notes:
- Ensure you keep all files in the same folder.
- Do not delete the "dist" folder or "inventory.db" file.
